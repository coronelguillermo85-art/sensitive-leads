# ⚡ sensitive-leads

## Descripción del Proyecto

Este repositorio contiene la estructura base y los activos de marca (`sensitive-logo-png`) para **Sensitive Leads**, una solución de front-end diseñada para centralizar y optimizar la captación de oportunidades comerciales (leads).

Como profesional con experiencia en **Relaciones Públicas (UNLaM)** y **Gestión de Servicios**, entiendo que la tecnología debe servir para mejorar la comunicación. Por eso, este proyecto no es solo código; es una herramienta de gestión diseñada para:

* **CX & KPI:** Mejorar la experiencia del usuario potencial y facilitar la medición de conversiones.
* **Automatización:** Sentar las bases para integraciones futuras con sistemas de automatización de marketing.

## Estado del Proyecto

Actualmente, el repositorio cuenta con:
* `index.html`: Estructura base del front-end.
* `sensitive-logo-png`: Activo de marca principal.

Este es el punto de partida para una herramienta de gestión de leads integrada con **IA aplicada** y flujos de trabajo eficientes.
